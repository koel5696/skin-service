<h1 align="center">Layer'd</h1>
<p align="center">
  피부 상태 기반 성분 · 제품 · 루틴 추천 시스템
</p>

## 1. Overview

- Layer'd 는 사용자의 피부 상태를 정량화하여 성분 → 제품 → 루틴까지 연결하는 추천 시스템입니다.
- 기존의 단순 피부 타입 기반 추천을 넘어 상태 벡터와 성분군 매칭을 기반으로 개인화된 결과를 제공합니다.

## 3. 주요 기능 소개

### 1). 피부 설문 기능
- 내 피부와 고민에 맞게 진단을 진행합니다.

### 2). 설문 기반 피부 상세 진단
- 설문을 바탕으로 내 피부에 대한 분석 결과를 제공합니다.
  - 내 핵심 피부 고민을 육각형 레이더 차트로 제공하여 한눈에 나의 피부 상태를 파악할 수 있습니다.
  - 지금 내가 스킨케어 관리 루틴에 필요한 요소 3가지를 제공합니다.
  - 케어가 필요한 부분에 대한 실제 핵심 화장품 원료 성분을 추천합니다.

### 3). 루틴 기능 제공
- 실제 피부 진단 결과를 바탕으로 사용자에 가장 최적화된 상품들로 직접 루틴을 구성합니다.
   - 루틴은 아침과 저녁으로 나뉘며, 각각 토너-앰플-크림-선크림, 토너-앰플-크림으로 구성합니다.
   - 아침에는 하루 일과가 시작되는 시간대이므로 가벼운 제품을, 저녁에는 내 피부 고민에 대한 핵심 성분이 고농도인 제품을 추천합니다.
   - 각 상품별로 제품 정보와 가격정보가 제공되며 링크를 바탕으로 외부 구매 사이트와 연결됩니다.
   - 루틴을 저장, 관리하여 나의 루틴 활용과 변화기록을 체계적으로 관리합니다.

### 4). 상품 정보 제공
- 루틴에서 추천되는 상품 이외에 내 피부와 맞는 제품을 더욱 다양하게 보여줍니다.

### 5). 마이페이지
- 저장된 루틴과 설문 분석 결과 재확인하고, 날짜 별로 관리할 수 있습니다.

## 3. Features

| 기능 | 설명 |
|------|------|
| 인증 | OAuth2 + JWT 기반 로그인 |
| 피부 진단 | 설문 기반 상태 분석 |
| 성분 추천 | Top3 성분군 및 추천 이유 제공 |
| 제품 추천 | Need-Supply 기반 점수 계산 |
| 루틴 생성 | 피부 상태 기반 스킨케어 루틴 |
